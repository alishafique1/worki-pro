## 3 — Build deploy artifacts (on `main`)

```bash
REACT_APP_API_URL=https://thehelper.ca ./deploy/build.sh
```
This runs `wasp build`, rebuilds the SPA into `deploy/dist/`, and **fails the build** if the bundle bakes the wrong `REACT_APP_API_URL` (guards the "Unexpected token '<'" class of bug). Note: `deploy/COOLIFY.md` step 4 still lists `api.thehelper.ca`; the canonical value per `build.sh` + the 2026-06-11 fix-review is the **same-origin** `https://thehelper.ca` (nginx proxies `/api`,`/auth`,`/operations`). Use the same-origin value unless the proxy design has changed.

```bash

git commit -m "build: update deployment artifacts [skip ci]"
git push origin main
```

C — Step wizard (category first, always)

Click "Request Help Today!" → goes to a multi-step wizard starting with category selection. Clear visual grid of categories. Once selected, advances to the existing request form. Best for mobile — no ambiguity. Scales perfectly as you add categories.

These can also be combined — e.g. A (modal) with C's step-by-step flow inside it.
